///////ARTHROPOD ///////

1 / Lancer l'installation de Arthropod en double cliquant sur le fichier .air
L'installation de AIR est nécessaire.

2/ Pour lancer Arthropod, cliquez sur le logo, il se placera devant toutes les fenetres normalement.

2/ Pour tracer dans la fenetre de Arthropod,il faut importer la classe Debug de Arthropod, puis l'utiliser de la manière suivante : 

Tracer un String : Debug.log("test");
Tracer un Object : Debug.object(myObject);
Tracer un Array : Debug.array(myArr);

ENJOY !
